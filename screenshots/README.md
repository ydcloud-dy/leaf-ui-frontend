# 项目截图目录

请将项目运行时的前端页面截图放置在此目录下。

## 建议的截图内容

1. **首页** - `home.png`
   - 展示文章列表和热门文章侧边栏

2. **文章详情** - `article-detail.png`
   - 展示 Markdown 渲染效果和代码高亮

3. **笔记分类** - `notes.png`
   - 展示笔记分类和章节目录功能

4. **文章归档** - `archive.png`
   - 展示按月份分组的归档页面

5. **网站统计** - `stats.png`
   - 展示统计数据和热门文章

6. **留言板** - `guestbook.png`
   - 展示留言和评论功能

7. **个人中心** - `profile.png`
   - 展示用户个人信息管理

8. **登录页面** - `login.png`
   - 展示登录/注册表单

## 如何添加截图到 README

在主 README.md 文件的"项目展示"部分添加：

```markdown
## 项目展示

### 首页
![首页](screenshots/home.png)

### 文章详情
![文章详情](screenshots/article-detail.png)

### 笔记分类
![笔记分类](screenshots/notes.png)

### 网站统计
![网站统计](screenshots/stats.png)
```
